# findneighbour-backend for an application that will help you find a neighbor to share an apartment with 

## used stack :

## ◼Node js

## ◼Express js

## ◼Mongo DB

## ◼date-fns

## ◼Socket io

## ◼Postgre SQL

## ◼Sequelize ORM

## ◼How To Start? :
1. Clone the repo</br>
```
$ git clone https://github.com/volodymyrpodufalyy/findneighbour-backend.git
```
2. Install NPM packages:
```
$ npm i
```
5. Start it!
```
$ npm start
```
